# Styled Icons Explorer (fork)

This repository is a fork of the original project:
https://github.com/styled-icons/website

## Purpose

The original demo site (**styled-icons.dev**) is currently unavailable.
This fork provides a minimal static icon explorer (search and icon list) for demo purposes.

https://lanam0r.github.io/website-icon/
